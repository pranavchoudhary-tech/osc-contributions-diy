## 👋 Welcome!

Are you new to open source and GitHub? This is the perfect place to start.

This project is designed to help beginners learn the standard open source workflow:
**Fork → Clone → Edit → Commit → Push → Pull Request**

No complex code required. Just pick an issue, make a small change, and open your first PR!

---

## 🚀 How to Contribute (Step by Step)

### Step 1 — Fork this repository
Click the **Fork** button at the top-right of this page.  
This creates a copy of this repo in your own GitHub account.

### Step 2 — Clone your fork
Open your terminal and run:
```bash
git clone https://github.com/<your-username>/osc-contributions-diy.git
cd osc-contributions-diy
```

### Step 3 — Create a new branch
Always work on a new branch, never directly on `main`:
```bash
git checkout -b your-name-or-change
```

### Step 4 — Make your change
Pick one of the open issues labeled `good first issue` and make the required change.

### Step 5 — Commit your change
```bash
git add .
git commit -m "Add: brief description of your change"
```

### Step 6 — Push to your fork
```bash
git push origin your-name-or-change
```

### Step 7 — Open a Pull Request
Go to your forked repo on GitHub. You'll see a **"Compare & pull request"** button — click it, fill in the PR template, and submit!
