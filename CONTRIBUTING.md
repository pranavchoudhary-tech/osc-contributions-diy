## How to Contribute?
1. Fork this repository
2. Clone your fork: git clone <your-fork-url>
3. Create a branch: git checkout -b your-name-change
4. Make your change
5. Commit: git commit -m "Add: your description"
6. Push: git push origin your-branch-name
7. Open a Pull Request back to this repo

## ⚠️ Important: Recurring Issues

Some issues in this repo are **permanent/recurring** — they are meant to stay
open forever so every new contributor can participate.

These include:
- "Add your name to the Contributors list"
- "Add a fun fact or quote"

For these issues, **do NOT write** `Closes #X` or `Fixes #X` in your PR.
Just write `Related to #X` instead so the issue stays open.
